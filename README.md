# 4-in-1 Web-Based Image Editor

This project is a web-based image editor that combines four powerful features:
1. **AI Image Enhancer**: Enhance images using AI models (mock implementation).
2. **Creative Editor**: Add filters, stickers, draw on images, and add text.
3. **Photo Adjustment**: Adjust brightness, contrast, saturation, highlights, and shadows.
4. **Layer-Based Editing**: Support for multiple layers, including moving, resizing, and blending modes.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Python Flask
- **Image Processing**: OpenCV, Pillow
